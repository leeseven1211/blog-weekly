## 🛠️ 工具推荐

**[Zed](https://zed.dev)**（代码编辑器）：由 Atom 原班人马打造的高性能代码编辑器，使用 Rust 编写，启动极快，原生支持协同编辑，是 VS Code 的有力竞争者。目前已开源。

**[Excalidraw](https://excalidraw.com)**（在线白板）：开源的手绘风格白板工具，无需注册即可使用，非常适合画架构图、流程图和头脑风暴。支持实时协作，数据可本地保存。

**[Bruno](https://www.usebruno.com)**（API 调试工具）：Postman 的开源替代品，配置文件直接存储为本地 Markdown 格式，天然适合 Git 版本管理，无需账号登录，完全离线可用。

---

## 🚀 有趣的项目

**[Hono](https://hono.dev)**（轻量级 Web 框架）：一个极简的 JavaScript/TypeScript Web 框架，可运行在 Cloudflare Workers、Deno、Bun、Node.js 等多种运行时上，API 设计优雅，性能出色，star 数量近期增长迅猛。

**[Docmost](https://github.com/docmost/docmost)**（开源知识库）：一个类 Notion/Confluence 的开源团队文档协作平台，支持实时协同编辑、权限管理、全文搜索，可自托管，适合替代昂贵的商业产品。

**[Screen Pipe](https://github.com/mediar-ai/screenpipe)**（本地 AI 记录工具）：一个在本地持续截屏并用 AI 分析的工具，让你随时能够"回放"自己的电脑使用记录，所有数据均存储在本地，隐私有保障。
